# glyph-zero

The origin template for all [glyph](https://github.com/dbriemann/glyph) powered blogs.

## Project Structure

```bash
├── config.toml
├── docs
│   └── _config.yml
├── LICENSE
├── Makefile
├── README.md
└── themes
    └── default
        ├── config.toml
        ├── index.mustache
        ├── issue.mustache
        ├── layout.mustache
        ├── style.css
        ├── tachyons.css
        └── tachyons.min.css
```

## Usage

1. Install [glyph](https://github.com/dbriemann/glyph). Make sure it's in your path.
2. Fork this repository.
